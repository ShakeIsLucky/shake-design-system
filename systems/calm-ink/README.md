## Signature qualities тАФ and how to keep them

1. **No frost, ever.** There is **no `backdrop-filter: blur`** in the entire system. The sticky header uses `saturate(105%)` only. Clarity, not glass, is the identity тАФ adding blur breaks it instantly.
2. **Paper grain** тАФ `body::before`, SVG `feTurbulence` at `--grain` (0.035) on `multiply`. Felt, not seen. Don't raise it.
3. **E-ink refresh sweep** тАФ `body::after`, a near-invisible vertical band (opacity 0 at rest; nudge it up to simulate a refresh).
4. **Breathing dot** тАФ `.dot-breathe`, a 6s `breathe` cycle (scale 1 тЖТ .82, halo expands + fades). The one piece of perpetual motion; keep it slow.
5. **Walking-pace reader** тАФ `.word` / `.word.lit` + `initReader`. Words light at **520ms** intervals, rest **4200ms**, dim, loop. These cadences *are* the calm тАФ leave them.
6. **Slow clock** тАФ `initClock` advances one minute every **3.2s**. Time passes, quietly.
7. **Self-drawing glyphs** тАФ `.glyph .draw` (stroke-dasharray) animates via `drawln` when a `.card` gets `.seen`.
8. **Hairlines + near-zero radius** тАФ dividers are 1px `--line`; corners are 2тАУ4px. No cards lift, nothing glows.

Motion gates on `prefers-reduced-motion` (`base.css`): the reader shows all words lit, the rest goes still.

## Using better fonts

Both picks are **free and drop-in anywhere** (Google Fonts + self-hostable), so they work whenever:

- **Stay on Google Fonts (default).** Wired in `base.css`. Alternates are listed there commented out тАФ **Source Serif 4** (crisper), **Spectral** (more literary), or **Newsreader** (the original sketch). Uncomment one, comment the default.
- **Self-host for offline / `file://`.** IBM Plex Mono is already in `~/Library/Fonts`; for the page, download Literata's `.woff2` (Google Fonts тЦ╕ Download, or fontsource.org), drop into `fonts/`, uncomment the `@font-face` block in `base.css`.
- **Premium (Adobe `lao8mse` тЖТ Freight Text Pro).** A natural fit for reading, but **won't render over `file://`** (Typekit domain allowlist). Only when served from an authorized http(s) host тАФ so not the "use anytime" answer here.

Change it in one place: the `--font-read` / `--font-mono` tokens in `tokens.css`, plus the `@import` in `base.css`.

## Accessibility
`prefers-reduced-motion: reduce` stops all animation/transition, reveals `.rise` content, shows the reader fully lit, and finishes drawn glyphs. Nothing depends on motion.

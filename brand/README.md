# Onda Studios — brand assets

The canonical mark for the Onda Studios design system: a sharp **compass star** (vertical-dominant
North-star) inside a horizontal **ellipse** (the "O" of ONDA).

| File | Use |
|---|---|
| `onda-star.svg` | Star only — the dingbat used inline in topbars/footers (`.star`, `currentColor`). |
| `onda-mark.svg` | Full mark — star inside the ellipse. Hero/footer lockups, favicons, README. |

Both are single-color via `currentColor`, so they inherit `--accent` (Onda gold) or `--ink`.

## Source of truth
Traced from the private Onda Brand Kit (`Logo 2025 PNG/`, `Newest Onda Star Logo PNG/`).
The official logo kit is **monochrome** (black/white) for
versatility. The brand's lived palette — confirmed by the studio's matte racing-green Ferrari 812
with a single gold stripe — is **deep racing-green + gold + black**, which the Racing-Green system
expresses via its tokens.

## Wordmark
`ONDA STUDIOS`, set in `var(--font-display)` (Freight Display Pro), all-caps, wide tracking.
Emphasis is **gold colour, never italic**.

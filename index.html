§ 01 — Typography

Serif italic for display, mono for everything else.

The body type is DepartureMono at 13px / 1.6 — Lefos's most distinctive move. Headlines and the occasional emphasis use PlantinNow italic (or EB Garamond italic where PlantinNow isn't licensed). Nav, labels, tags, captions, table cells, and code blocks all run mono.

§ 01 / 04
type · scale

The reading column caps at 720 px (~64ch at 13px mono), nested inside a 1080 px outer frame. Tables and figures may opt into the wider frame.

Body links are underlined sentence case, accent-brass on hover. Strong text is just colour-shifted to ivory — no bold weight needed.

§ 03 — Visualisations

Arc-leaning playful inside, palette-locked outside.

Mermaid uses theme: 'base' with brass / sage / terracotta / parchment classDef tints. Edges render brass; node text reads ivory and re-skins automatically with the page theme. Hover a figure to see the shake grace note — the only animation that ships by default.
